# NotebookLM Исследование

## Обзор
Эксперименты с NotebookLM для извлечения знаний из загруженных источников.

## Структура
- `session_[timestamp]/` - отдельные сессии вопросов-ответов
- `themes/` - тематические папки по мере выявления
- `insights/` - ключевые инсайты и выводы
